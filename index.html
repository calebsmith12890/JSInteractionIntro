Controls -------------------------------

Aim Cannon: Cursor
Fire Cannon: Left Click


Gameplay -------------------------------

Primary Colors: Red, Yellow, Blue
Secondary: Purple, Green, Orange

16 Defending Orbs are randomly assigned
Either Primary or Secondary colors.

Attacking Orbs are shot from the cannon,
but can only start as Primary colors.

If an attacking orb hits a defender with
a primary color, and the colors match,
the defender dies.

If the defender is Primary, but does not
match, the attackers color is added to the
defender.

If the defender is a secondary color and,
that color contains the attackers primary
color, that color is removed from the color
of the defender.
Attackers bounce off of the top border,
but disappear if they pass the bottom.
